# GoTours Georgia

A full-stack web application for booking airport transfers and car rentals in Georgia. Built with React and Express.js, featuring multi-language support (EN, ES, RU, KA) and an admin dashboard.

## Tech Stack

**Frontend:** React 19, Vite, Tailwind CSS, React Router, i18next, Google Maps API

**Backend:** Express.js, MongoDB, Passport.js (OAuth), JWT

## Getting Started

```bash
# Clone
git clone https://github.com/Cxvarisorka/gotours.ge.git

# Install & run server
cd server && npm install && npm run dev

# Install & run client
cd client && npm install && npm run dev
```

See `.env.example` files in `client/` and `server/` for required environment variables.
